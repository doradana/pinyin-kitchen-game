# 拼音廚房使用說明

雙擊 `start_website.bat` 即可啟動網站。啟動後會出現一個黑色伺服器視窗，請保持它開著，然後用瀏覽器開啟：

`http://127.0.0.1:8055/index.html`

同一台電腦可以開多個分頁模擬不同玩家。若瀏覽器顯示無法連線，請確認黑色伺服器視窗仍然開著，再重新整理頁面。

## 學生端

1. 選擇小組並按「加入廚房」。
2. 一組最多 5 個人。
3. 選擇自己的主要廚具：鍋子或砧板。每位玩家都會自帶一個盤子。
4. 點選或拖曳食材到自己的主要廚具或盤子。
5. 拼音 + 聲調放進鍋子，可以煮成中文字。
6. 中文字放到砧板，可以切成拼音 + 聲調。
7. 每位玩家都有自己的菜單，把自己菜單指定的中文字放到自己的盤子，即可得分。
8. 食材往左或往右滑出畫面，可以傳給同組上一位或下一位組員。左滑送出時，接收者畫面會從右邊滑進該食材；右滑送出時，接收者畫面會從左邊滑進。
9. 食材和廚具都可以在桌面上自由拖動位置。

## 老師端

1. 切換到「老師」。
2. 設定題目，格式為：`繁體,簡體,拼音,聲調`，例如：`媽,妈,ma,1`。
3. 選擇答案字體：繁體或簡體。
4. 設定組數與回合時間。
5. 按「儲存設定」後，再按「開始回合」。
6. 小組監控區會顯示每組分數、玩家、鍋子、砧板、盤子狀態。

舊格式 `中文字,拼音,聲調` 仍可使用。若沒有提供簡體，遊戲會優先使用內建常見字轉換表，找不到對應時會沿用原字。

目前這版已包含本機多人同步伺服器；同一個網址開在多個裝置或分頁中，可以同步玩家、食材傳遞與老師監控。
